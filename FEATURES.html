✨ Características Principales

🎮 Mecánicas TD Clásicas

Sistema completo de Tower Defense con colocación de torres, oleadas de enemigos y gestión de recursos.

🎨 Gráficos ASCII Animados

Representación visual completamente ASCII con renderizado en tiempo real. Enemigos, torres y proyectiles animados.

♾️ Niveles Infinitos

Dificultad progresiva sin límite. Cada nivel incrementa número, velocidad y vida de enemigos.

🔊 Audio Dinámico

Web Audio API con generación procedural de sonidos para disparos, impactos, explosiones y más.

🏢 Arquitectura Modular

Código separado en 6 módulos independientes y bien documentados. Fácil de extender y modificar.

📱 GitHub Pages Ready

Sin dependencias, sin backend. Funciona abriendo index.html o desplegando directamente en GitHub Pages.

⌨️ Controles Intuitivos

Soporte para teclado (1-4) y ratón. Botones UI para control visual. Tecla espacio para pausar.

🎯 4 Tipos de Torres

Básica, Rápida, Potente e Hielo. Cada una con mecánicas únicas y costos diferentes.

👾 3 Tipos de Enemigos

Normal, Rápido y Pesado. Aparecen según el nivel con vida y velocidad variable.

📊 Sistema de Puntuación

Puntos basados en enemigos derrotados, nivel actual y oleada completada.

💾 Estado Persistente

Sesiones de juego completas con pausas, reinicio y reintentos sin perder progreso.

🚀 Cómo Usar

1. Ejecutar Localmente

Simplemente abre index.html en tu navegador, o usa un servidor local:

python -m http.server 8000
# Luego accede a http://localhost:8000

2. Desplegar en GitHub Pages

  1. Crea un repositorio en GitHub llamado tower-defense-ascii
  2. Sube los archivos del proyecto
  3. Ve a Settings → Pages → Source: main/root
  4. ¡Listo! Tu juego estará en https://tu-usuario.github.io/tower-defense-ascii/
